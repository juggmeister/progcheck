## Summary

This website was built using modern web technologies including React 18, TypeScript, Vite, and Tailwind CSS. The UI is composed primarily of shadcn/ui components built on Radix UI primitives, with extensive use of Framer Motion for animations. The backend is powered by Supabase, providing PostgreSQL database, authentication, and real-time capabilities.

All dependencies are open-source and licensed under permissive licenses (primarily MIT, Apache 2.0, or ISC).
